# Archivos Públicos y Multimedia

Esta es la carpeta `public`. Todos los archivos que coloques aquí (como tus **imágenes**, modelos 3D, íconos o recursos multimedia) se servirán automáticamente desde la raíz del sitio web. 

- Si tienes una imagen `mi-foto.png` guardada en esta carpeta, dentro de tu HTML y tus componentes de React (como `Sidebar.jsx`, `Viewer3D.jsx`) puedes referenciarla fácilmente escribiendo literalmente el nombre desde la raíz del dominio, así:
`<img src="/mi-foto.png" />`
